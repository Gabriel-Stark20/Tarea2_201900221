Tutorial de JAVA



¿Cómo crear una clase y una instancia de clase?



- Paso 1:

Se crea una clase nueva en nuestro proyecto, una Java Class, con un click derecho en el paquete de nuetsro proyecto.










- Paso 2:

Se crean las variables inicializadas para poder construir nuestro metodo, en este caso las variables son private int tamaño=0; private String color="";. Nótese que también se creo el contructor de la clase, el constructor es public void telefonoo(){}










- Paso 3:

Se crea un método, en este caso se creará un método para imprimir el color y tamaño de el teléfono, el método es imprimir();














- Paso 4:

Para poder acceder a las variables, ya que en pincipio son privadas por el modificador de acceso private no son accesibles, pero con los metodos de acceso Getters y Setters podemos acceder.














- Paso 5:

Para instanciar la clase se crea un objeto que llevará la clase que queremos. Se remarcó en color rojo la instancia de clase como tal.













- Paso 6:

Se imprime nuestro método y se observa el resultado.
